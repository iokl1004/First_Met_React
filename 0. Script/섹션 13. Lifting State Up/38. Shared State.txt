38. Shared State

여러 개의 컴포넌트들 사이에서 state를 공유하는 방법
리액트로 개발하다 보면 하나의 데이터를 여러 개의 컴포넌트에서 표현해야 하는 경우가 종종 생김.
이러한 경우에 각 컴포넌트의 스테이트에서 데이터를 각각 보관하는 것이 아니라 가장 가까운 공통된 부모 컴포넌트의 스테이를 공유해서 사용하는 것이 더 효율적임.

Shared state는 말 그대로 공유된 state를 의미함.
자식 컴포넌트들이 가장 가까운 공통된 부모 컴포넌트의 스테이트를 공유해서 사용하는것.
