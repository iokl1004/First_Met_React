Node.js? 는 자바스크립트로 네트워크 애플리케이션을 개발 할 수 있게끔 해주는 환경.
npm (Node Package Manager)

1. node.js와 npm 설치를 위해 하기의 링크 접속
https://nodejs.org/en/

3. 설치 후, cmd창 Open 후 아래의 명령어 입력
